# Competency Based Test — PDF Upload Directory

Upload your chapterwise MCQ PDFs here, organized by subject.

## Folder Structure

```
competency/
├── mathematics/    ← Upload Maths PDFs here
├── physics/        ← Upload Physics PDFs here
├── chemistry/      ← Upload Chemistry PDFs here
└── biology/        ← Upload Biology PDFs here
```

## PDF Format

Each PDF should contain MCQs with:

- Question text
- 4 options (A/B/C/D)
- Correct answer
- Year of paper
- Marks per question (1/2/3/4)
